Zombie Survival

Zombie Survival is a singleplayer wave survival game in which the player tries to survive as long as they can while purchasing upgrades in the shop to help them continue. The game is currently written by WappyCoder in scratch, but it is working on being converted to Phaser JS which should help it run better.


Controls:
W,A,S,D: Movement
P: Open Shop
E: Energy Shot
R: Roll
Space Bar: Chainsaw or RiotShield
1: Omega Bomb
2: Hydra Beam
